# SIMATA Resources

This folder contains all the resources needed for the SIMATA ride-hailing app.

## Structure

```
resources/
├── images/         # App images (backgrounds, banners, illustrations)
├── animations/     # Lottie animation files (.json)
├── fonts/          # Custom font files (.ttf, .otf)
├── icons/          # Custom SVG/PNG icons not covered by vector icons
├── sounds/         # Sound effects and notification tones
└── maps/           # Map style configs and custom markers
```

## Required Resources

### Images
- `onboarding_1.png` — Welcome illustration
- `onboarding_2.png` — Ride booking illustration
- `onboarding_3.png` — Safety illustration
- `map_placeholder.png` — Map fallback image
- `promo_banner.png` — Promotional banner

### Animations (Lottie)
- `car_moving.json` — Car animation for ride tracking
- `loading.json` — Loading spinner animation
- `success.json` — Ride completion celebration
- `searching.json` — Driver search radar animation

### Fonts
- `Inter-Regular.ttf`
- `Inter-Medium.ttf`
- `Inter-SemiBold.ttf`
- `Inter-Bold.ttf`

### Icons
- `marker_pickup.png` — Custom pickup location marker
- `marker_dropoff.png` — Custom dropoff location marker
- `car_top_view.png` — Top-view car icon for map

### Maps
- `dark_map_style.json` — Dark mode map styling config
- `light_map_style.json` — Light mode map styling config
